## 🖼️ 图片替换说明

当前页面使用 Unsplash 免费图作为占位图。
如需替换为真实品牌图片，请按以下路径替换：

### 替换方法
1. 打开 `pages/index-xxx.html`
2. 搜索 `<img src="https://images.unsplash.com/...`
3. 替换为你的图片URL或相对路径

### 推荐替换图片清单

| 页面 | 用途 | 建议尺寸 | 数量 |
|------|------|---------|------|
| 品牌故事 | Hero封面 | 1200×520px | 1张 |
| 品牌故事 | 创始人/茶园图 | 800×450px | 1张 |
| 十大香型 | 各香型茶汤/干茶图 | 600×600px | 10张 |
| 冲泡指南 | 茶具/冲泡过程图 | 800×450px | 3-5张 |
| 买茶 | 各系列产品图 | 800×600px | 6-8张 |
| 体验预约 | 茶园/制茶/品鉴场景图 | 800×450px | 3张 |

### 自主拍摄建议
- **产品图**：茶汤特写（白底或自然光）+ 干茶条索 + 冲泡过程
- **场景图**：凤凰山茶园（云雾缭绕）+ 制茶工艺（摇青/烘焙）+ 工夫茶冲泡
- **人像**（可选）：制茶匠人工作场景、消费者品茶表情
